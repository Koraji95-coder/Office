# Class Notes

Drop study material here when you want EE Mentor to use it as local context.

Good file types:
- `.md`
- `.txt`
- `.docx`
- `.pdf`
- `.pptx`

Good material:
- lecture notes
- lab writeups
- formula sheets
- worked examples
- standards excerpts
- project-specific study notes

After adding files, click `Refresh Office` so the knowledge library rescans them.
