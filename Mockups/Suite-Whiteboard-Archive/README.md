# Suite Whiteboard Archive

This folder preserves the former Suite whiteboard source after it was removed from the Suite app.

It is an archived React implementation, not a wired DailyDesk feature. The source still reflects its original Suite dependencies such as shared system components, auth/notification hooks, and the Supabase `whiteboards` table.

Archived roots:

- `src/features/whiteboard`
- `src/routes/developer/labs/whiteboard`

If you decide to revive this in Office later, treat this folder as source reference for a deliberate Office-native port rather than drop-in runnable code.
